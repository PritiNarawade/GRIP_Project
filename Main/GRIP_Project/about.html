#Task 1

  • Create a simple dynamic website which has the following specs.

  • Start with creating a dummy data in database for upto 10 customers. Database options:Mysql,Mongo,Postgres,etc. Customers table have basic fields such as name,email,current balance etc. Transfers table will record all transfers happend.
